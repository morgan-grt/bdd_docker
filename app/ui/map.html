Carte de visualisation de données

Voici notre carte de visualisation de données

Nous pouvons retrouver une carte avec les différentes villes sélectionnées :

  • Bordeaux
  • Caen
  • Nice
  • Paris
  • Hambourg
  • Nuremberg
  • Sarrebruck
  • Barcelone
  • Madrid
  • Seville

La carte est dessinée à partir d'un fichier geoJSON et les villes positionnées à l'aide de leur
longitude et latitude

Certaines valeurs ne sont pas disponibles ce qui correspond à une valeur 0 sur les graphes.
Il est donc possible d'avoir des chutes à 0°C

La carte est interractive :

Vous pouvez accéder aux données d'une ville soi en cliquant dans le menu à droite
soi en cliquant directement sur un point de la carte

Chaque ville représente un lien pour créer un graphe spécifique à la ville
Ce graphe peut-être étendu sur la durée, nous pouvons aussi choisir la variables entre tavg, tmin et tmax
Nous pouvons retrouver ce graphe avec le tableau

ps : pour cacher ce panel il suffit de cliquer dessus
