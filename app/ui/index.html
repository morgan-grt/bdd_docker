Bienvenue sur notre de site de visualisation de données


Voici quelques informations concernant le site :


Notre thème est la météo, nous utilisons les données météorologiques de 10 villes d'Europe (des stations météorologiques) :

  • 4 en France ( Bordeaux, Caen, Nice et Paris )
  • 3 en Allemagne ( Hambourg, Nuremberg et Sarrebruck )
  • 3 en Espagne ( Barcelone, Madrid et Seville )

Les données se situent sur la période 2000-2021 (~ 70000 lignes) et ont été récupérées depuis la NOAA qui est l'agence américaine responsable de l'étude de l'océan et de l'atmosphère.

La NOAA met à disposition une énorme quantité de données météo en libre accès et sous différents formats (du json, du csv, du grib : un format spécial pour les données météo...).

Les données ont été récupérées au format csv, traitées à l'aide de Microsoft Excel puis converties en JSON

Nous avons extrait pour chaque jour 10 données par ville :

  • Le pays
  • La ville
  • La date
  • La latitude de la station
  • La longitude de la station
  • Les précipitations
  • La quantité de neige
  • La température moyenne
  • La température maximale
  • La température minimale

Nous pouvons retrouver toutes ces données dans le tableau

Les données affichées sur le site ont été auparavant chargées dans une base non traditionnelle mongo à l'aide d'un fichier JSON

Pour faire tourner ce site interractif, il y a 4 conteneurs déployés, pour mongo, mongo-express, graphql et le site

Le site tourne sur un serveur express nodejs

Graphql permet d'envoyer des requêtes vers une base de données et de récupérer des données en retour, il tourne sur un serveur Apollo nodejs

Les graphes interractifs présents sur le site sont dessinés à l'aide de d3.js, une librairie qui permet l'analyse, le traitement et l'utilisation de données notamment en graphe

Chaque page possède une documentation, cliquez sur le bouton "documentation" à droite pour l'afficher et sur la doc elle-même pour la cacher

Bonne visite!
